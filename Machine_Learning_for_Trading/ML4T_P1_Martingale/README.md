# Project 1: Martingale Strategy Simulation

## Overview
This project explores the Martingale betting strategy using an American Roulette wheel. The objective is to evaluate its performance under different bankroll conditions and understand the probabilistic concepts involved.

## Files
- **`martingale.py`**: Python script implementing the Martingale strategy simulation.
- **`martingale_report.pdf`**: Detailed report of the project, including methodology, results, and analysis.
